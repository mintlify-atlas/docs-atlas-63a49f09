## Rethinking browser navigation

Nook is a modern macOS browser built for the way you actually browse the web. With vertical tabs in a persistent sidebar, you get a natural, uncluttered workflow that scales from a handful of tabs to hundreds.

## Why Nook?

### Sidebar-first navigation

Vertical tabs feel natural and uncluttered. Unlike horizontal tabs that get cramped and unreadable with more than a dozen open, Nook's sidebar scales beautifully. See exactly what you have open, organize tabs into spaces and folders, and never lose track of your workflow.

### Built for performance

Nook is optimized with modern macOS APIs for speed and low memory use. The browser uses:

- **WKWebView** for native WebKit rendering
- **Swift 6** with strict concurrency for thread safety
- **Lazy webview loading** to minimize memory footprint
- **Intelligent tab unloading** to keep inactive tabs from consuming resources

### Minimal, modern interface

Every pixel is focused on content, not chrome. The interface adapts to your needs:

- Customizable sidebar position (left or right)
- Optional top-bar address view or sidebar-integrated URL entry
- Configurable materials and visual effects
- Command palette for keyboard-driven navigation

### Privacy and control

- **Multiple profiles** with isolated data stores for work, personal, and testing
- **Incognito mode** with non-persistent storage
- **Cross-site tracking protection** (optional)
- **Extension support** via WKWebExtension (macOS 15.4+)

## System requirements

<Note>
Nook requires **macOS 15.5 (Sequoia)** or later. Extension support requires macOS 15.4+.
</Note>

- **Architecture**: Universal (Apple Silicon and Intel)
- **Built with**: Xcode, Swift 6, SwiftUI
- **License**: GPL-3.0 (with exceptions for third-party libraries)
